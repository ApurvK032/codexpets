# Spark Starter

Spark is the starter custom pet draft.

- `pet.json`: manifest for the pet picker.
- `spritesheet.png`: original demo `1536x1872` working sprite sheet.
- `spritesheet-guide.png`: labeled grid reference. Do not use it as the final pet image.

Install by copying `pet.json` and the finished sprite sheet into:

```text
%USERPROFILE%\.codex\pets\spark
```
